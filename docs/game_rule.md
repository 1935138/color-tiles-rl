## 게임
Color Tiles

### 보드 구성
- 전체 보드 크기: 23 × 15 (총 345칸)
- 사용되는 타일 색상: 총 10종
  - White / Pink / Blue / Sky Blue / Green / Orange / Yellow / Purple / Brown / Red
  - 각 색상별 20개씩 배치 (총 200개)
- 빈칸: 145개 (타일 이동 및 연결 경로 역할)

### 게임 규칙

**제한시간:** 120초

**조작 방법:**
- 빈칸을 클릭하여 타일을 제거한다.

**타일 제거 조건:**
1. 클릭한 빈칸을 기준으로 상/하/좌/우 4방향을 확인한다.
2. 각 방향에서 빈칸으로 연결된 경로를 따라 가장 먼저 만나는 타일을 찾는다.
   - 중간에 다른 타일이 있으면 그 방향은 막힌 것으로 간주한다.
3. 4방향에서 찾은 타일 중 같은 색상의 타일이 2개 이상이면 해당 타일들이 모두 제거된다.
   - 예: 같은 색 3개가 보이면 3개 모두 제거

**점수 시스템:**
- 타일 1개 제거 시 1점 획득

**패널티:**
- 같은 색상 타일이 1개만 있거나 없는 빈칸을 클릭하면 시간 10초 감소

### 게임 종료 조건

**클리어:**
- 모든 타일(200개)을 제거하면 게임 클리어

**실패:**
- 제한시간(120초) 초과
- 남은 타일이 있지만 연결 가능한 같은 색상 타일 조합이 더 이상 없는 경우
